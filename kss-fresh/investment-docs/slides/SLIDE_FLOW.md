## 네비게이션 정보

- 각 슬라이드에서 클릭 또는 → 키로 다음 슬라이드로 이동
- ← 키로 이전 슬라이드로 이동
- ESC 키로 전체화면 종료
- 모든 슬라이드는 위의 순서대로 연결되어 있음
