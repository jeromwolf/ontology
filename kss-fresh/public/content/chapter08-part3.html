Phase 1

도메인 모델링

45분
🧬 의료 도메인 모델링 워크샵
클래스 계층 구조 설계
🏥 MedicalEntity
😷 Symptom
🌡️ Fever
😴 Fatigue
🤕 Pain
🧠 Headache
❤️ ChestPain
🦠 Disease
🫁 RespiratoryDisease
😷 COVID19
🫁 Pneumonia
❤️ CardiovascularDisease
💊 Treatment
💉 Medication
🏥 Surgery

📝 자동 생성된 OWL 코드
# 의료 온톨로지 클래스 정의

Declaration(Class(:MedicalEntity))
Declaration(Class(:Symptom))
Declaration(Class(:Disease))
Declaration(Class(:Treatment))

# 계층 구조
SubClassOf(:Symptom :MedicalEntity)
SubClassOf(:Disease :MedicalEntity)
SubClassOf(:Treatment :MedicalEntity)

# 증상 하위 클래스
SubClassOf(:Fever :Symptom)
SubClassOf(:Fatigue :Symptom)
SubClassOf(:Pain :Symptom)
SubClassOf(:Headache :Pain)
SubClassOf(:ChestPain :Pain)

# 질병 하위 클래스
SubClassOf(:RespiratoryDisease :Disease)
SubClassOf(:CardiovascularDisease :Disease)
SubClassOf(:COVID19 :RespiratoryDisease)
SubClassOf(:Pneumonia :RespiratoryDisease)

# 치료 하위 클래스
SubClassOf(:Medication :Treatment)
SubClassOf(:Surgery :Treatment)
SubClassOf(:Therapy :Treatment)
Phase 2

관계 및 추론 규칙 정의

60분
🔗 의료 관계 설계자
핵심 Object Properties
🔍 hasSymptom
Domain: Disease Range: Symptom Inverse: isSymptomOf
예제: COVID19 hasSymptom Fever
💊 treatedBy
Domain: Disease Range: Treatment Inverse: treats
예제: Pneumonia treatedBy Antibiotics
⚠️ contraindicated
Domain: Medication Range: Medication Type: Symmetric
예제: Aspirin contraindicated Warfarin
📊 hasSeverity
Domain: Symptom ∪ Disease Range: SeverityLevel Type: Functional
예제: HighFever hasSeverity Severe
🧠 지능형 추론 규칙
규칙 1: 질병 진단 추론 
# 규칙: 특정 증상 조합으로 질병 추론
Class: COVID19Candidate
    EquivalentTo: Person and (
        hasSymptom some Fever and
        hasSymptom some (Cough or DryThroat) and
        hasSymptom some Fatigue and
        not (hasSymptom some (Rash or Nausea))
    )

# SWRL 규칙
Person(?p) ∧ hasSymptom(?p, ?fever) ∧ Fever(?fever) ∧
hasSymptom(?p, ?cough) ∧ Cough(?cough) ∧
hasSymptom(?p, ?fatigue) ∧ Fatigue(?fatigue)
→ possibleDiagnosis(?p, COVID19)

설명: 발열, 기침, 피로 증상이 있으면서 발진이나 메스꺼움이 없는 환자를 COVID19 후보로 분류

신뢰도:
85%
규칙 2: 약물 상호작용 경고 
# 규칙: 위험한 약물 조합 감지
Class: DangerousCombination
    EquivalentTo: Treatment and (
        consistsOf some (Medication and contraindicated some Medication)
    )

# 예시: 혈액 희석제 조합
Class: AnticoagulantCombo
    EquivalentTo: Treatment and (
        consistsOf some BloodThinner and
        consistsOf some AspirngradientColorin
    )

SubClassOf(AnticoagulantCombo DangerousCombination)

설명: 금기사항이 있는 약물 조합을 자동으로 감지하고 경고
