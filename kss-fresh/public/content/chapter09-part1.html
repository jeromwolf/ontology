🎯 추론의 근본 원리

⬇️

연역 추론 (Deductive Reasoning)

확실성: 100%

정의: 일반적 원리에서 특수한 경우를 도출하는 논리적 추론

특징: 전제가 참이면 결론도 반드시 참

🧪 연역 추론 시뮬레이터
대전제
소전제
결론
따라서 고래는 심장이 있다 ✓ 논리적으로 유효
형식 논리 표현
∀x (Mammal(x) → HasHeart(x))    // 모든 포유류는 심장이 있다
Mammal(whale)                     // 고래는 포유류다
─────────────────────────────────
∴ HasHeart(whale)                // 따라서 고래는 심장이 있다
⬆️

귀납 추론 (Inductive Reasoning)

확실성: 가능성

정의: 특수한 관찰에서 일반적 원리를 도출하는 추론

특징: 결론이 전제보다 더 많은 정보를 담음

가설 추론 (Abductive Reasoning)

확실성: 추측

정의: 관찰된 현상을 가장 잘 설명하는 가설을 추론

특징: "최선의 설명으로의 추론"

📈

단조성 (Monotonicity)

시스템 특성

정의: 새로운 정보가 추가되어도 기존 결론이 변하지 않는 성질

특징: Classical Logic은 단조적, Human Reasoning은 비단조적

⚖️ 단조 vs 비단조 추론 비교
🔒 단조 추론 (Classical Logic)
초기 지식: 새는 날 수 있다
관찰: 펭귄은 새다
결론: 펭귄은 날 수 있다
새 정보: 펭귄은 날지 못한다
문제: 모순 발생! 🚫
🔄 비단조 추론 (Human-like)
초기 가정: 새는 보통 날 수 있다
관찰: 펭귄은 새다
임시 결론: 펭귄은 아마 날 수 있다
새 정보: 펭귄은 날지 못한다
수정 결론: 펭귄은 예외적인 새 ✅

9.2 Description Logic: 추론의 수학적 기초

🧮 Description Logic이란?

📐

Definition

Description Logic (DL)은 개념, 역할, 개체들 사이의 관계를 형식적으로 표현하고 추론하기 위한 논리 체계입니다.

🏷️ Concepts (개념) 클래스, 집합
🔗 Roles (역할) 관계, 속성
👤 Individuals (개체) 인스턴스

🎛️ DL 표현력 탐색기

복잡도 레벨 선택

🌱 Basic Level: ALC (Attributive Language with Complements)

복잡도: EXPTIME
구문 구성기
기본 구조
A 개념
r 역할
⊤ 전체
⊥ 공집합
논리 연산자
⊓ 교집합
⊔ 합집합
¬ 부정
양화사
∃r.C 존재
∀r.C 전칭
표현식 만들기
여기에 DL 구문 요소를 드래그하세요
예제 표현식
Person ⊓ ∃hasChild.Person 부모 (사람이면서 자식이 있는)
Parent ⊓ ∀hasChild.HappyPerson 행복한 부모 (모든 자식이 행복한 부모)
Person ⊓ ¬∃eats.Meat 채식주의자 (고기를 먹지 않는 사람)
🧠 의미 해석기
현재 표현식
Person ⊓ ∃hasChild.Person
의미 해석
1️⃣
Person: 사람 개념의 집합
2️⃣
∃hasChild.Person: 적어도 한 명의 사람 자식을 가진 개체들의 집합
3️⃣
⊓: 두 집합의 교집합
💡
결과: 사람이면서 동시에 자식을 가진 개체들 = 부모
